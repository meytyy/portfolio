# Meytyy Portfolio

Персональный сайт-портфолио на React + TypeScript + Vite + Tailwind.

## Локальный запуск

```bash
npm install
npm run dev
```

После этого открой адрес из терминала, обычно:

```bash
http://localhost:5173
```

## Сборка

```bash
npm run build
npm run preview
```

## Деплой

### Netlify
- Build command: `npm run build`
- Publish directory: `dist`

### Vercel
- Framework Preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`
